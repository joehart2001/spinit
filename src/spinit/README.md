# spinit Source Layout

This README documents the internal package structure under `src/spinit`.
It is intended for contributors and maintainers.

## Scope

- Public entry points are in the top-level package files (`__init__.py`, `api.py`, `config.py`).
- Implementation is grouped into focused subpackages: `graph`, `chemistry`, `seeding`, and `output`.

## Directory: `src/spinit/`

| File | Purpose |
|---|---|
| `__init__.py` | Package-level exports for common user-facing functions (`assign_initial_magnetic_moments`, config helpers, reporting helpers). |
| `api.py` | End-to-end orchestration pipeline: graph build, ring extraction, feature build, scoring, and moment assignment strategy selection. |
| `config.py` | Default thresholds/weights and config utilities (pair cutoffs, merge helpers, element detection). |
| `README.md` | Internal maintainer-oriented map of the source tree (this file). |

## Directory: `src/spinit/graph/`

| File | Purpose |
|---|---|
| `__init__.py` | Re-exports graph, geometry, and topology helpers for convenient imports. |
| `bond_graph.py` | Bond graph construction and local ring/cycle utilities. |
| `graph_ops.py` | PBC-aware graph + MIC bond geometry attachment and low-level graph access helpers. |
| `geometry.py` | Vector and angle utilities built on bond vectors. |
| `topology.py` | Ring-derived topology feature helpers and structure-level topology summaries. |

## Directory: `src/spinit/chemistry/`

| File | Purpose |
|---|---|
| `__init__.py` | Re-exports chemistry feature/hybridization/motif functions. |
| `features.py` | Builds per-atom feature dictionaries by combining graph, geometry, motif, and hybridization signals. |
| `hybridization.py` | Hybridization-like classification and sp2 strain/pi-alignment analysis helpers. |
| `motifs.py` | Connectivity-aware local motif classification for C/H/O/N and fallback handling for other elements. |

## Directory: `src/spinit/seeding/`

| File | Purpose |
|---|---|
| `__init__.py` | Re-exports scoring and magnetic moment assignment strategy functions. |
| `scoring.py` | Intrinsic + neighbor-modifier magnetic scoring and candidate-site selection logic. |
| `assignment.py` | Converts candidate scores into signed initial moments (FM, AFM-cluster heuristic, random-candidate strategies). |

## Directory: `src/spinit/output/`

| File | Purpose |
|---|---|
| `__init__.py` | Re-exports reporting helpers. |
| `reporting.py` | Human-readable summaries and optional table construction for feature/candidate inspection. |

## Generated Directories

These directories are generated by Python and are not source modules:

- `src/spinit/__pycache__/`
- `src/spinit/graph/__pycache__/`
- `src/spinit/chemistry/__pycache__/`
- `src/spinit/seeding/__pycache__/`
- `src/spinit/output/__pycache__/`

They contain compiled bytecode (`.pyc`) and can be ignored for code review.
